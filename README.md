# 🏙️ My District (Мой Район)
> **Authentic urban exploration game developed entirely on an Android tablet.**

---

## 📸 Overview
Проект воссоздания атмосферы родного района в 3D. Исследование, интерактивность и уют панельных домов — всё это создается без использования ПК, используя только мобильные мощности.

### 🛠 Tech Stack (Mobile-Only Workflow)
| Stage | Tool | Environment |
| :--- | :--- | :--- |
| **Engine** | **Godot 4.3** | Android Native (Mobile Renderer) |
| **Modeling** | **Blender 4.5 LTS** | Windows Emulator (GameHub) |
| **Coding** | **Acode** | Android Native (High-speed editor) |
| **VCS** | **Git / Termux** | SSH-access & Shell management |
| **Assets** | **Sketchfab** | External 3D resource integration |

---

## 📱 Hardware Setup
Мой мобильный "воркстейшн", на котором крутится весь этот проект:
* **Tablet:** Xiaomi Pad 5 (Xiaomi Smart Pen 2nd Gen) — *Основная разработка*
* **Primary Phone:** Poco X6 Pro — *Тестирование производительности*
* **Secondary Phone:** Poco F3 — *Дебаг и дополнительные тесты*
* **Peripherals:** Keyboard & Mouse connected via Hub.

## 🚀 How to build
Так как проект в Godot 4:
1. Клонируй репозиторий: `git clone git@github.com:PavloVoiat/MyDistrict.git`
2. Импортируй `project.godot` в Godot Engine 4.3+.
3. Наслаждайся (или помогай с кодом!).
